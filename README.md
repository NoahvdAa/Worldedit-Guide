# WorldEdit Guide

A guide to using the Minecraft plugin/mod Worldedit.

All screen-captures on this guide are made on a [Spigot](https://spigotmc.org/) 1.15 server with the [regular WorldEdit plugin](https://dev.bukkit.org/projects/worldedit/) and the [WorldEditSUI Visualizer plugin](https://www.spigotmc.org/resources/worldeditsui-visualize-your-selection.60726/).

Note: This guide is not affiliated, associated, authorized, endorsed by, or in any way officially connected with the WorldEdit team.
